# EventBus 

[![Build Status](https://travis-ci.org/com-nilportugues/com.nilportugues.eventbus.svg?branch=master)](https://travis-ci.org/com-nilportugues/com.nilportugues.eventbus)

## Installation

Add in your `pom.xml` file the jitpack.io repositories:

```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```
  
Now add the package as a dependency: 

```xml
<dependencies>		
  <dependency>
    <groupId>com.nilportugues</groupId>
    <artifactId>eventbus</artifactId>
    <version>${eventbus.version}</version>
  </dependency>
</dependencies>  
```
